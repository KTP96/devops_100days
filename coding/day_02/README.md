## Lists

- A lists stores multiple values in one variable.
- **Ex**:
  ```
  servers=["web-01","web-02","db-01"]
  instance_ids=["i-123","i-456","i-789"]
  ```
- **Accessing**
  ```
  print(servers[0])
  print(servers[1])
  ```
- **Looping**:
  ```
  for server in servers:
  print(f"Checking {server}")
  ```
- **Adding Items:**
  ```
  servers.append("web-1")
  ```
